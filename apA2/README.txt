
- JDK 17
- JavaFX 21.0.1
- Junit 5

How to install:

Download the project
Navigate to the project directory using: -cd in cmd
Compile and build the project using: javac -cp .:path/to/javafx-sdk-X.X.X/lib/* application/MainGUI.java
Run the project using: java --module-path path/to/javafx-sdk-X.X.X --add-modules javafx.controls application.Main

How to Use the program:

Register a new user or use the test data of test,test as username and password
Add posts, view posts, edit profile
Register as a VIP user and use the VIP features
